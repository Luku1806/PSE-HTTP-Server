\subsection{Detailed Description}
This file provides all functions for processing the incoming requests, from reading everything into a struct, to generating an answer. 

In general this should be used like\+:


\begin{DoxyEnumerate}
\item Process raw message by calling parse\+Request to save all informations in a request\+\_\+struct
\item Put the struct into generate\+Response to build a response struct
\item Put the response\+\_\+struct into http\+Response\+To\+String to get the response as string 
\end{DoxyEnumerate}
