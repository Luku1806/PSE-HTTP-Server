Detailed Description

Macro Definition Documentation

◆ HTPASSWD_PATH

#define HTPASSWD_PATH   "/usr/local/webroot/htpasswd"

The path where the htpasswd file, with all the http credentials in it is located.

This file can be created by the command: "htpasswd -s -c htpasswd [username]" The server only supports SHA1 encrypted credentials, so keep in mind to set the -s flag when creating credentials with htpasswd.

◆ STATUS_SITE_PATH

#define STATUS_SITE_PATH   "/usr/local/webroot/status/"

The path where all status pages lie. Each status page is named after its status. Each status has its own site, and any html file with a fitting name can be used.

For example a page for status 505 is named: "505.html"
